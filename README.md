# Weather App

## Description
Used OpenWeather API to retrieve weather data for cities for this weather dashboard app that runs in the browser and features dynamically updated HTML and CSS.

## Tech Used:
* HTML
* JavaScript
* Bootstrap
* CSS
* Open Weather API
